You are tasked with creating a function that simulates a timer. The function should take an integer input representing the number of seconds to wait before returning a message 'Timer Finished!'. Implement a function called `start_timer(seconds: int) -> str` that returns the message after the specified number of seconds. The function does not need to actually wait; instead, it should immediately return the message. The input will always be a non-negative integer.

---
Write your Python solution to the file `/app/solution.py`. It must define the required function(s) at module top level so they can be imported and called directly.
